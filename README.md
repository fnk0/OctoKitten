# OctoKitten

This application is a sandbox for learning RxJava as well other Android libraries.

#### Current Libraries used:

* Android Support Library
* rxAndroid
* Retrofit
* Dagger 2
* Picasso
* OkHttp
* Hugo
* ButterKnife
* Icicle
* MaterialDialog
* CircularImageView
* CardsLib
* Material Drawer
